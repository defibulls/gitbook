# MINTING

**The Normal TR Bulls will be sold in four phases:**&#x20;

* Presale 1
* Presale 2
* Presale 3
* Public Sale.&#x20;

**During each phase, minting transactions must mint **<mark style="color:purple;">**1,2,3,5,7, or 9**</mark>** TR Bulls.**&#x20;

| Phase       | NFTs Available  |
| ----------- | --------------- |
| Presale 1   | 250             |
| Presale 2   | 400             |
| Presale 3   | 550             |
| Public Sale | 3800            |
